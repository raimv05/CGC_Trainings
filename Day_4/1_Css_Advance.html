Theory
What Is the CSS Box Model?
Every HTML element is a rectangular box — the model describes its 4 layers
Imagine a mobile phone inside a delivery box.
Phone
= Content
The actual element: text, image, button
Thermocol
= Padding
Inner cushion — space inside the element
Cardboard
= Border
The visible edge/boundary of the element
Space Around Box
= Margin
Outer gap — pushes other elements away
box model diagram — draw this on the board
+--------------------------------+ | Margin | | +------------------------+ | | | Border | | | | +------------------+ | | | | | Padding | | | | | | +------------+ | | | | | | | Content | | | | | | | +------------+ | | | | | +------------------+ | | | +------------------------+ | +--------------------------------+
Memory trick: Padding is a Pillow (inside, between content and wall). Margin is a Moat (outside, keeps other things away).
Why it matters: Nearly every layout problem in CSS — elements overlapping, unexpected gaps, things not lining up — comes down to misunderstanding the box model. Master this and you fix 70% of CSS bugs.

Explain
Property-by-Property Explainer
Click any row to expand — each property explained with what it does visually
background: #2563eb
background — fills the content + padding area
Sets the blue fill. The background colour extends into the padding area but stops at the border.
This is why padding makes the box look bigger — the background colour fills the padding region too. Change to #dc2626 for red, #059669 for green. Try it!
width: 300px
width — controls only the content area
Sets the content area to 300px wide. Padding and border are added on top of this.
Total visible width = width + padding-left + padding-right + border-left + border-right = 300 + 30 + 30 + 5 + 5 = 370px. This surprises beginners — box-sizing: border-box fixes this (we'll learn that in Hour 2).
padding: 30px
padding — inner cushion on all 4 sides
Adds 30px of space between the text "This is a Box Model Example" and the border edge.
Shorthand values: padding: 30px = all sides. padding: 10px 20px = top/bottom 10px, left/right 20px. padding: 10px 20px 30px 40px = top, right, bottom, left (clockwise). Background colour fills padding.
border: 5px solid black
border — the visible edge around the element
A 5px thick solid black line drawn around the outside of the padding.
Border shorthand = width style color. Style options: solid (continuous line) · dashed (dashes) · dotted (dots) · double (two lines). Width in px. Color = any CSS colour value. Try border: 3px dashed red.
margin: 40px
margin — outer space, pushes other elements away
Adds 40px of transparent space outside the border on all sides.
Margin is always transparent — you never see it, but it takes up space. Use margin: 0 auto to center a block element horizontally (works when width is set). Same shorthand rules as padding.
border-radius: 10px
border-radius — rounds the corners
Rounds the 4 corners by 10px. 0 = sharp square corners. 50% = perfect circle (if width = height).
Modern UI design uses border-radius almost everywhere — buttons, cards, profile pictures. Try border-radius: 20px for very rounded, border-radius: 50% for a circle (change width and height to same value). border-radius: 0 is sharp.

Teach
Padding vs Margin — The Key Difference
The most commonly confused concept in CSS — visualised side by side
Padding — inner space
.box { padding:20px; }
Padding region (background fills this)
Box content
Content moved inward. The green area is all part of the element — background colour fills it.
Padding in SkillHub
header { padding:20px; }
/* Text gets breathing room inside */
Margin — outer space
.box { margin:20px; }
Margin region (always transparent)
Box content
Whole box moved outward. The yellow area is transparent — pushes other elements away.
Margin in SkillHub
h2 { margin-bottom:20px; }
/* Gap between h2 and next element */
one-line rule
Padding = space inside the box
Background colour fills it
Margin = space outside the box
Always transparent

Reference
Box Model Properties Reference
Complete guide — keep this open while building SkillHub
PropertyWhat it doesExampleValues / Options
padding Inner spacing — between content and border. Background fills this area. padding: 20px 20px all · 10px 20px top/bottom left/right · 10px 20px 10px 20px T R B L
margin Outer spacing — transparent gap outside the border. Pushes other elements away. margin: 40px Same shorthand as padding. 0 auto = centres block horizontally
border Visible edge drawn around the element (outside padding). border: 5px solid black Width · Style (solid dashed dotted double) · Color
border-radius Rounds the corners of the border box. border-radius: 10px 0 sharp · 10px slight round · 50% circle
width Width of the content area only (not including padding/border by default). width: 300px 300px · 50% · auto (fills parent)
box-sizing Controls what "width" includes. border-box = width includes padding + border. box-sizing: border-box content-box (default) · border-box (modern practice) — taught in Hour 2
Shorthand direction: When you see 4 values like padding: 10px 20px 10px 20px, they go clockwise from top — Top · Right · Bottom · Left. Remember: TRouBLe.
