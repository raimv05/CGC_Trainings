Topic 1
What is a Website?
Ask students — what do these have in common?
familiar examples
Google
Facebook
Amazon
Instagram
❓ What do all these have in common?
✅ They are all Web Applications
They run in a browser, interact with servers, and store data — not just static pages.
key insight
A web application is a program that runs inside a browser. It has a frontend the user sees, a backend that handles logic, and a database that stores data.
Topic 2
How does the Internet Work?
Food delivery analogy → request & response
the analogy
Customer
= Browser
Restaurant
= Server
Kitchen Storage
= Database
request–response flow
1
You type www.amazon.com
Browser captures the URL
2
Browser sends a Request
Like calling the restaurant to place an order
3
Server receives the request
Amazon's server processes what you asked for
4
Database fetches products
Like the kitchen getting ingredients from storage
5
Response sent back → page displayed 🎉
Food delivered to your door!
