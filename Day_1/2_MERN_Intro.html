Step 1
Install VS Code
Visual Studio Code — your primary code editor
what is vs code?
Visual Studio Code
A free, open-source code editor made by Microsoft. It's the most popular editor for web developers worldwide — lightweight, fast, and packed with extensions.
Extensions
Add features like React snippets, GitLens, Prettier
Built-in Terminal
Run Node, Git, and npm commands without switching apps
Syntax Highlighting
Color-codes HTML, CSS, JS, JSX for easy reading
Download: Go to code.visualstudio.com → Click the big blue Download button → Run the installer → Done!
recommended extensions for mern
ES7+ React Snippets Prettier ESLint GitLens Thunder Client
Step 2
Install Node.js
The runtime that lets JavaScript run on the server
what is node.js?
Node.js lets you run JavaScript outside the browser — on your computer and servers. Without it, Express and npm (our package manager) won't work. Think of it as the engine that powers the backend of SkillHub.
verify node installation
Terminal
$node -v
v22.x.x
$npm -v
10.x.x
What is Node.js?
JavaScript runtime built on Chrome's V8 engine. Runs JS code server-side — powers our Express backend.
What is npm?
Node Package Manager — comes bundled with Node. Used to install React, Express, and every library we need.
Tip: Always download the LTS (Long-Term Support) version from nodejs.org — it's the most stable and recommended for projects.
Step 3
Install Git
Version control — track, save, and collaborate on code
what is git?
Git is a version control system — it takes snapshots of your code over time. If you break something, you can always roll back. Think of it as an unlimited undo button for your project.
verify git installation
Terminal
$git --version
git version 2.x.x
Branches
Work on features without breaking main code
History
See every change made to every file
Collaboration
Multiple devs work on the same project
Download: Visit git-scm.com → Download for your OS → Install with default settings → Verify with git --version.

Step 6
Create Git Repository
Initialise version control for the project
command & output
Terminal — inside SkillHub-LMS/
$git init
Initialized empty Git repository in .../SkillHub-LMS/.git/
what just happened?
A hidden .git/ folder was created inside SkillHub-LMS
Git is now watching your project — ready to take snapshots
Every git commit you make is saved forever
You can roll back to any commit at any time
git init
Creates a new empty repository in the current folder. Run this only once per project.
git clone <url>
Used when a repository already exists online (GitHub). We'll use this later when pushing to GitHub.

Step 8
First Commit
Save your project's first snapshot with Git
commands in order
Terminal — inside SkillHub-LMS/
$git add .
# Stages all files — tells Git "watch these"
$git commit -m "Initial project setup"
[main (root-commit)] Initial project setup 1 file changed, 4 insertions(+) create mode 100644 README.md
git add .
Stage
Select which files Git should include in the next snapshot
git commit -m
Commit
Save the snapshot with a message describing the change
git log
History
See all commits made so far — try this after committing!
