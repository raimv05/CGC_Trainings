Day 19 · Part 1
Registration & Password Hashing
Never store a plain-text password — type one below and hash it
routes/auth.js
app.post("/register", async(req,res)=>{
  const bcrypt = require("bcrypt");
  const hashedPassword =
    await bcrypt.hash(req.body.password, 10);
  // save hashedPassword to MongoDB
});
Try It — hash a password
Never store
123456
Store instead
$2b$10$..........................
If a database leaks and passwords were stored in plain text, every user account is instantly compromised. Hashing makes the stored value practically irreversible.

Day 19 · Part 2
Login System
Try logging in — a registered user is preloaded below
routes/auth.js
app.post("/login", async(req,res)=>{
  const user = await User.findOne({ email: req.body.email });
  const valid = await bcrypt.compare(req.body.password, user.password);
  if(valid) res.json({ message: "Login Successful" });
  else res.json({ message: "Invalid Credentials" });
});
